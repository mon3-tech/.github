## Sobre a MON 3

A **MON 3** desenvolve infraestrutura e experiências digitais para pagamentos internacionais, gestão de liquidez e contas digitais.

Nosso foco é simplificar operações financeiras complexas com tecnologia robusta, integrações confiáveis e uma experiência de produto moderna.

---

## O que estamos construindo

- Pagamentos cross-border
- Infraestrutura de liquidez
- Contas digitais e operações financeiras
- Integrações com provedores de pagamento e serviços financeiros
- Automação operacional para produtos fintech
- Experiências web e mobile para fluxos financeiros
